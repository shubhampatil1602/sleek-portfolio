# AWS Essentials for Web Developers

AWS is the most widely adopted cloud platform. It offers services like compute, storage, and database hosting.

## Key Services

- **EC2**: Virtual servers in the cloud.
- **S3**: Scalable object storage.
- **Lambda**: Serverless functions.
- **RDS**: Managed databases.

## Example: Hosting a Static Site on S3

1. Upload files to an S3 bucket.
2. Enable static website hosting.
3. Set public access permissions.
4. Use Route 53 or CloudFront for a custom domain.
